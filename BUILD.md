# Building the game

## Requirements

- **Love2D 11.4 Mysterious Mysteries** or later
  - It _might_ work with older versions, but **no support is provided.**
- **Just**
  - Available at <https://github.com/casey/just>
- **MoonScript**
  - Available at <https://moonscript.org/>

## Running the game

```sh
just run
```
